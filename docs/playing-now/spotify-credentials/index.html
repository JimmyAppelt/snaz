Spotify API Credentials

Why do I need these

In order to receive album info and it’s art for a player, authentication with the Spotify API is required.

Note

This counts for all players except iTunes

Browse to the developer section of Spotify and login

https://developer.spotify.com/ 

Create an app

Copy your credentials and paste them in the Snaz settings
