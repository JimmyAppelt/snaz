Note

Album and album art are query-ed from the Spotify API (will respond with the first match, if any) for all players except iTunes, you are able to get the album and album art you set within iTunes. In order to receive the album art Authentication with the Spotify API is required.

Spotify Credentials

In order to receive album info and it’s art, authentication with the Spotify API is required. (for all players except iTunes)

Json file

  • Json data for track info
  • The directory to output this file can be chosen in the menu File → Settings → Playing Now
  • Example code for dynamic playing now content with the json file

Save track information in separate text files

This option is available in the menu File → Settings → Playing Now

When enabled, TrackArtist.txt, TrackSong.txt and TrackAlbum.txt will be available in the TextFiles folder on the next track.
