Time

Display current system time with customizable formatting for overlays and streams

Date

Show current system date with various format options for different regions and preferences

Countdown

Live countdown timer to a specific date/time
(Example: Stream starts in: 2h 30m 12s)

Chrono Down

Countdown timer that decreases from a set value, perfect for break timers or event durations

Chrono Up

Stopwatch-style timer that counts up from zero or a specified starting value

Twitch

Real-time Twitch viewer count display
(module omitted in v1.12.7.0, use v1.12.6.0)

Textline Changer

Dynamically cycle through multiple text lines in a single output file for rotating messages

System Info

Real-time system monitoring displaying CPU usage, RAM consumption, active processes, and network speeds

Dynamic Files

Generate custom text or image files linked to online data sources for live content updates

Playing Now

Currently playing music information including artist, album, and track details
(module omitted in v1.12.7.0, use v1.12.6.0)
